### Object<Transformation>

Following the structure used by [magnum](https://magnum.graphics) we represent each OBject as:
- an entity in a SceneGraph
- a container of features as to how this entity can be used

#### SceneGraphNode
As a node in a SceneGraph each object has transformation data, which can be dirtied and cleaned


#### AbstractObject<EmbeddingType>
A basic AbstractObject contains a list of Features attached to the object
